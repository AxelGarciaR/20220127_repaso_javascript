Investigacion sobre los metodos de array

Método reduce: El método reduce se utiliza para reducir los elementos de un array a un solo valor, aplicando una función acumuladora. Toma una función de reducción y un valor inicial, y luego itera sobre cada elemento del array, actualizando el acumulador con el resultado de la función aplicada a cada elemento.

Método filter: El método filter se utiliza para crear un nuevo array con todos los elementos que cumplan cierta condición. Toma una función de filtrado y retorna un nuevo array que contiene únicamente los elementos del array original que pasan dicha condición.
